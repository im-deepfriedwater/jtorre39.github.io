Maze Builder

Using JavaScript, a touch of jQuery, and a dash of HTML/CSS, Maze Builder exhibits the use of direct manipulation to allow users to build their own maze on devices with web browsers that support touch.

KeyFrame Tweener

KeyFrame Tweener is a from-first-principles JavaScript project meant to create a tool that calculates the in-between frames from a JSON file of key frames. Demonstrated by my own animation titled, "Emotional Cartharsis", we prefer the art to speak for itself around here.
